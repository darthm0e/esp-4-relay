# esp-4-relay
Based on ESP_WEBSERVER_HTTP11 by Stefan Thesen 04/2015


Modified 04-2017

4-Relay Module controll over WiFi
Tested on WeMos D1 mini

### used Pins:

5V

GND

D1

D2

D3

D4 *switches also ESP Blue LED so maybe use oter Pin



## LCD Version:

### i2c:

D1 - SCL

D2 - SDA


### Relays:

D5 - Relay1

D6 - Relay2

D7 - Relay3

D4 - Relay4
